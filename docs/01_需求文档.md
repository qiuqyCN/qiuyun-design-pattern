# 秋云设计模式 - 需求文档

## 1. 项目概述

### 1.1 项目名称

**秋云设计模式** (Qiuyun Design Patterns)

### 1.2 项目目标

创建一个基于 Astro + Vue 3 + shadcn-vue + TailwindCSS 的纯前端设计模式学习网站，通过丰富的动画效果和交互式体验，帮助开发者直观理解 23 种经典设计模式。

### 1.3 技术栈

- **框架**: Astro 5.x (静态站点生成)
- **UI 框架**: Vue 3 + shadcn-vue
- **样式**: TailwindCSS 4.x
- **动画**: GSAP / Framer Motion / Vue Transitions
- **图标**: Lucide Vue
- **代码高亮**: Shiki

***

**各区块详细设计：**

**① Hero Section（首屏）**
- **背景动画**：
  - 漂浮的云朵粒子（符合"秋云"主题）
  - 设计模式关系图的动态连线（若隐若现）
  - 渐变色背景：天空蓝到云朵白
- **品牌展示**：
  - Logo：云朵 + 代码符号的组合图标
  - 主标题："秋云设计模式"（大字号，渐变色文字）
  - 副标题："Qiuyun Design Patterns"
  - 标语：打字机效果逐字显示 "23种经典设计模式，动画辅助学习"
- **操作按钮**：
  - 主按钮："开始探索"（带脉冲动画）
  - 次按钮："查看学习进度"
- **滚动提示**：底部弹跳箭头引导用户向下滚动

**② 学习概览 Stats Section**
- **四宫格数据卡片**：
  - 已学习：0/23（环形进度条）
  - 连续学习：0天（火焰图标）
  - 完成进度：0%（条形进度）
  - 学习时长：0分钟（时钟图标）
- **交互动画**：数字滚动递增效果
- **点击跳转**：进入详细统计页面

**③ 三种分类 Category Section**
- **三列卡片布局**：
  - 创建型（🏗️）：5种模式 - 关注对象创建
  - 结构型（🧩）：7种模式 - 关注类/对象组合
  - 行为型（🎭）：11种模式 - 关注对象间通信
- **卡片效果**：
  - 悬停 3D 翻转或上浮阴影
  - 显示该分类的学习进度（如：已学 2/5）
  - 分类主色调区分（创建型-蓝、结构型-绿、行为型-紫）
- **点击交互**：展开该分类的模式列表或跳转到列表页

**④ 热门模式 Popular Section**
- **横向滚动列表**：展示最常用的 6 个设计模式
- **卡片内容**：
  - 模式名称 + 使用频率星级（⭐⭐⭐⭐⭐）
  - 简短描述（一行）
  - 难度标签
  - [学习] 按钮
- **交互**：鼠标悬停暂停滚动，可左右拖拽

**⑤ 今日推荐 Recommendation Section**
- **智能推荐卡片**：
  - 基于用户学习进度推荐下一个该学的模式
  - 或随机推荐一个未学习的模式
- **内容展示**：
  - 模式名称 + 图标
  - 核心定义（一句话）
  - 操作按钮：[立即学习] [换一个] [收藏]
- **学习提示**：根据进度给出学习建议
  - "你已经学习了 3 个创建型模式，建议接下来学习结构型模式"
  - "连续学习 5 天，继续保持！"

**⑥ 学习路径 Path Section**
- **可视化路线图**：
  - 横向时间线展示推荐学习顺序

  - 当前推荐显示 👉
  - 未学习的显示 ⬜
- **交互**：点击节点跳转到对应模式详情

**⑦ 特色功能 Features Section**
- **三列展示核心卖点**：

- **滚动动画**：各区块进入视口时的淡入/滑入效果
- **视差滚动**：Hero 背景与内容不同速滚动
- **导航栏**：
  - 首屏隐藏，滚动后滑入显示
  - 当前所在区块高亮
- **快捷操作**：
  - 右下角固定按钮：回到顶部、快速搜索
- **响应式适配**：
  - 移动端：单列布局，分类卡片改为横向滑动
  - 平板端：两列布局

**首屏加载优化：**
- Hero 背景使用 CSS 动画而非视频
- 其他区块懒加载
- 首屏关键 CSS 内联

**各模块详细设计：**

**① 模式概述 (Overview)**
- **头部信息区**: 模式名称（中文大标题 + 英文副标题）+ 分类标签（带颜色）+ 难度徽章（⭐⭐⭐）+ 使用频率标签（高/中/低）
- **一句话定义**: 用通俗语言描述模式核心思想
- **适用场景**: 图标 + 文字列表（如：✅ 需要解耦时、✅ 对象创建复杂时）
- **优缺点对比**: 左右分栏表格，优点用绿色，缺点用橙色

**② 结构图解 (Structure) ⭐ 核心交互区**
- **可视化画布**:
  - 使用 SVG/Canvas 绘制类图
