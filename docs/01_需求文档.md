# 秋云设计模式 - 需求文档

## 1. 项目概述

### 1.1 项目名称

**秋云设计模式** (Qiuyun Design Patterns)

### 1.2 项目目标

创建一个基于 Astro + Vue 3 + TailwindCSS 的纯前端设计模式学习网站，通过丰富的动画效果和交互式体验，帮助开发者直观理解 23 种经典设计模式。

### 1.3 技术栈

- **框架**: Astro 5.x (静态站点生成)
- **UI 框架**: Vue 3 (Composition API)
- **样式**: TailwindCSS 4.x
- **动画**: Vue Transitions / CSS Animations
- **代码高亮**: Shiki
- **数据持久化**: LocalStorage

### 1.4 已实现的主要功能

- ✅ 23种设计模式完整内容（创建型5个、结构型7个、行为型11个）
- ✅ 23个交互式动画演示
- ✅ 首页、学习路径、统计、关于、前置知识等多个页面
- ✅ 学习进度统计与追踪
- ✅ Markdown导出功能
- ✅ 暗黑模式支持
- ✅ 响应式设计

***

## 3. 学习统计功能

### 3.1 学习统计数据结构

```typescript
interface LearningStatistics {
  // 总体进度
  totalPatterns: number;        // 23
  learnedPatterns: number;      // 已学习数量
  completedPercentage: number;  // 完成百分比

  // 连续学习
  streakDays: number;           // 当前连续学习天数
  longestStreak: number;        // 最长连续学习记录
  lastLearnedDate: string;      // 最后学习日期 (ISO 格式)

  // 学习时长
  totalLearningTime: number;    // 总学习时长（分钟）
  patternLearningTime: Record<string, number>; // 每个模式的学习时长

  // 分类统计
  categoryProgress: {
    creational: { total: 5; learned: number };
    structural: { total: 7; learned: number };
    behavioral: { total: 11; learned: number };
  };

  // 学习记录
  recentLearned: string[];      // 最近学习的模式 ID 列表（最多10个）
  dailyRecords: DailyRecord[];   // 每日学习记录
}

interface DailyRecord {
  date: string;                 // 日期 (YYYY-MM-DD)
  patternsLearned: string[];    // 当天学习的模式
  learningTime: number;         // 当天学习时长（分钟）
}
```

### 3.2 学习计时功能

**LearningTimer 组件**：
- 无 UI，后台静默运行
- 页面加载时开始计时
- 每 5 分钟自动记录一次学习时长
- 页面切换/隐藏时自动保存已学习时长
- 页面关闭或跳转时保存剩余时长
- 数据保存在 localStorage

### 3.3 数据持久化

- **存储方式**: LocalStorage
- **存储键**:
  - `qiuyun-stats`: 学习统计数据
  - `qiuyun-learned-patterns`: 已学习模式集合
- **已学习模式准确统计**: 使用 Set 数据结构记录，不受 10 个限制
